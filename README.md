# NAC — Nine Acre Corner

Henry Bonchek's weekly college radio show on 88.1 FM.

A curated archive of every show's playlist with album art, listening links, and pattern analysis spanning decades of rock, indie, and alternative music.

## Stack
- Static HTML/CSS/JS (single page)
- Chart.js for data visualizations
- Spotify data embedded at build time
- Deployed on Vercel

## Refresh Data
Track data is embedded directly in `public/index.html`. To update with new shows, re-run the Spotify import script and rebuild.
